GitHub+Hexo 搭建个人网站

参考链接
https://zhuanlan.zhihu.com/p/32957389

1 安装hexo
$ cd
$ npm install -g hexo
$ hexo init 1. 创建独立博客项目文件夹
$ npm install 2. 安装依赖包
$ npm install hexo-deployer-git –save 3. 确保git部署
$ hexo g $ hexo s 4.本地查看

2 关联git
安装git插件
npm install hexo-deployer-git –save

git的配置，修改_config.yml文件:
deploy:
  - type: git
    repository: git@github.com:funybaby/blog.git
    branch: master

你要确保你本地能git push到服务端，所以需要配置本地的ssh github私钥

3 hexo 常用命令
hexo clean //清理
hexo new “postName” //新建文章
hexo new page “pageName” //新建页面
hexo g //生成静态页面至public目录
hexo server //开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy //将.deploy目录部署到GitHub

4 添加 SSH key 到 GitHub
ls ~/.ssh 1 检查SSH keys是否存在
ssh-keygen -t rsa -C “your_email@example.com“ 2 生成新的 SSH key
cat id_rsa.pub 3 将 SSH key 添加到 GitHub 中
4 进入GitHub –> Settings –> SSH and GPG keys –> NEW SSH key
